Python Fundamentals

Learn the most powerful programming language

🎯 11 Practice Problems

1 Print "Hello World"
Display your first message in Python.
print("Hello World!")
print("My first Python program!")
2 Create variable
Store data using variables.
name = "John"
age = 25
print("Name:", name)
print("Age:", age)
3 Basic arithmetic
Perform simple calculations.
a = 10
b = 3
print("Add:", a + b)
print("Subtract:", a - b)
print("Multiply:", a * b)
print("Divide:", a / b)
4 If statement
Make decisions with conditions.
age = 20
if age >= 18:
    print("Adult")
else:
    print("Minor")
5 For loop 1-10
Loop through numbers 1 to 10.
for i in range(1, 11):
    print(i)
6 Simple function
Create reusable code blocks.
def greet(name):
    print("Hello", name)

greet("Alice")
greet("Bob")
7 User input
Get input from user.
name = input("Enter your name: ")
print("Hi", name, "welcome!")
8 List creation
Create and display a list.
fruits = ["apple", "banana", "orange"]
print("Fruits:", fruits)
print("First fruit:", fruits[0])
9 Basic list loop
Loop through list items.
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)
10 Simple calculator
Build a basic calculator.
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
print("Add:", num1 + num2)
print("Subtract:", num1 - num2)
print("Multiply:", num1 * num2)
print("Divide:", num1 / num2)
11 🎉 COMBO CHALLENGE: Complete Python Program
Combine ALL 10 Python skills into one complete program!
print("=== Welcome to Python Calculator ===")

# Variables and Input
name = input("Enter your name: ")
print("Hello", name, "!")

# Numbers 1-10 loop
print(" Counting 1 to 10:")
for i in range(1, 11):
    print(i)

# Simple calculator
print(" === Simple Calculator ===")
num1 = float(input("First number: "))
num2 = float(input("Second number: "))
print("Result:", num1 + num2, num1 - num2, num1 * num2)

# Age checker function
def check_age(age):
    if age >= 18:
        return "Adult"
    else:
        return "Minor"

age = int(input("Enter your age: "))
status = check_age(age)
print("Status:", status)

# Fruit list
