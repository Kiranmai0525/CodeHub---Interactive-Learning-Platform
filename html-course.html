HTML Fundamentals

Master the building blocks of web development

🎯 11 Practice Problems

1 Create a webpage title
Add a title that appears in the browser tab using the title tag.
<!DOCTYPE html>
<html>
<head>
<title>My First Webpage</title>
</head>
<body>
</body>
</html>
2 Add a main heading (h1)
Create the largest heading on your webpage.
<h1>Welcome to My Website</h1>
3 Create an ordered list (1,2,3)
Make a numbered list with 3 items.
<ol>
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ol>
4 Create an unordered list (bullets)
Make a bulleted list with 3 items.
<ul>
<li>Apple</li>
<li>Banana</li>
<li>Orange</li>
</ul>
5 Add a paragraph with text
Add a block of text to your webpage.
<p>This is a paragraph with some sample text.</p>
6 Create a simple link
Make a clickable link to Google.
<a href="https://google.com">Visit Google</a>
7 Add an image placeholder
Add a placeholder image to your page.
<img src="https://via.placeholder.com/300x200" alt="Sample Image">
8 Create headings h2 + h3
Add medium and small headings.
<h2>About Me</h2>
<h3>My Skills</h3>
9 Make a simple form (name input)
Create an input field for name.
<form>
<input type="text" placeholder="Enter your name">
</form>
10 Create a button
Add a clickable button.
<button>Click Me!</button>
11 🎉 COMBO CHALLENGE: Complete webpage
Combine ALL 10 skills into one complete webpage!
<!DOCTYPE html>
<html>
<head>
<title>My Portfolio</title>
</head>
<body>
<h1>John Doe</h1>
<h2>About Me</h2>
<p>Web developer learning HTML</p>
<h3>Skills</h3>
<ul>
<li>HTML</li>
<li>CSS</li>
</ul>
<a href="#contact">Contact</a>
<img src="https://via.placeholder.com/300x200" alt="Photo">
<form>
<input type="text" placeholder="Name">
<button>Submit</button>
</form>
</body>
</html>
