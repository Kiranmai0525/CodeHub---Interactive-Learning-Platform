CSS Fundamentals

Style your web pages like a pro

🎯 11 Practice Problems

1 Change background color
Set a background color for any HTML element.
<!DOCTYPE html>
<html>
<head>
<style>
body { background-color: #f0f0f0; }
</style>
</head>
<body>
<p>This has gray background!</p>
</body>
</html>
2 Change text color
Change the color of text content.
<h1 style="color: #333333;">Dark text</h1>
<p style="color: #FF6B6B;">Pink text</p>
3 Make text bold
Make any text appear bold.
<p style="font-weight: bold;">This text is bold!</p>
<p>This text is normal.</p>
4 Change font size
Increase or decrease text size.
<h1 style="font-size: 36px;">Big Heading</h1>
<p style="font-size: 12px;">Small text</p>
5 Add padding to element
Add internal spacing inside an element.
<div style="padding: 20px; background: #ddd;">
  This box has padding!
</div>
6 Center text
Horizontally center text content.
<div style="text-align: center;">
  <h2>Centered Text</h2>
  <p>This is centered!</p>
</div>
7 Change button color
Style a button with custom colors.
<button style="background-color: #4CAF50; color: white; padding: 10px;">
  Green Button
</button>
8 Add border to element
Create a visible border around elements.
<div style="border: 3px solid #ddd; padding: 15px;">
  This has a border!
</div>
9 Make text uppercase
Transform text to all uppercase letters.
<p style="text-transform: uppercase;">hello world</p>
<p>normal text</p>
10 Change margin
Control external spacing around elements.
<div style="margin: 30px; background: #eee; padding: 10px;">
  Box with margin
</div>
11 🎉 COMBO CHALLENGE: Complete styled page
Combine ALL 10 CSS skills into one beautiful webpage!
<!DOCTYPE html>
<html>
<head>
<title>My Portfolio</title>
<style>
body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 20px;
}
.container {
  margin: 40px auto;
  max-width: 800px;
}
h1 {
  color: #2c3e50;
  font-size: 36px;
  text-align: center;
}
.card {
  background: white;
  padding: 30px;
  border: 3px solid #3498db;
  border-radius: 10px;
}
button {
  background-color: #e74c3c;
  color: white;
  padding: 12px 24px;
  border: none;
  font-weight: bold;
}
.skills {
  text-transform: uppercase;
  font-weight: bold;
}
</style>
</head>
<body>
<div class="container">
<h1>John Doe</h1>
<div class="card">
<p>Full Stack Developer</p>
<div class="skills">HTML | CSS | JavaScript</div>
<button>Hire Me!</button>
</div>
</div>
</body>
</html>
